# 600 · IA Native

Esta sección existe para que cualquier modelo de lenguaje — Claude, ChatGPT, Gemini, Kimi, Meta AI, Grok, o el que venga después — pueda entender el proyecto Somos del Pueblo de punta a punta y ayudar a un colaborador humano a trabajar en él, sin depender de que ese colaborador ya conozca todo el contexto de memoria.

No es documentación técnica de una API. Es contexto de proyecto, pensado para pegarse directamente en la conversación con cualquier LLM al inicio de una sesión de trabajo.

## Cómo usar esta sección

Si sos un colaborador (humano) que va a pedirle ayuda a una IA para trabajar en algo de Somos del Pueblo — un texto, un guion, una pieza de contenido, una decisión de producto — pegale a la IA el contenido de `contexto-proyecto.md` antes de pedirle la tarea. Si la tarea depende de un dato específico (precio, nombre de plan, estructura societaria), verificalo también contra `datos-de-referencia.md` — es la fuente de verdad, y se actualiza cada vez que algo cambia.

## Archivos de esta sección

- **`contexto-proyecto.md`** — quién es Somos del Pueblo, la filosofía completa, el modelo de negocio, para qué existe.
- **`datos-de-referencia.md`** — los datos concretos que no deben inventarse ni aproximarse: precios, nombres de planes, estructura societaria, plataformas.
- **`guia-colaboracion-ia.md`** — cómo debería comportarse una IA ayudando en este proyecto: tono, límites, temas confidenciales, qué no prometer todavía.

## Por qué existe esta sección aparte

Porque el resto del brand book está escrito para personas — con el tono, la voz y el ritmo con el que queremos que se sienta el proyecto. Esta sección está escrita para que cualquier IA la lea rápido y no se pierda en la forma en su prisa por entender el fondo. Ambas cosas — el tono humano del resto del libro y la claridad directa de esta sección — persiguen lo mismo: que el proyecto se entienda bien, sin importar quién o qué lo esté leyendo.
